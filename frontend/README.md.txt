# VayoMitra: AI Companion for Aging with Dignity

**VayoMitra** is an AI-powered companion platform for senior citizens that supports:
- 💊 Medication reminders
- 🗣️ Voice-based emotional interaction
- 📞 Local service finder
- 📚 Memory support & storytelling (Katha Sathi)
- 🔒 Privacy-first sentiment analysis

This project includes:
- React frontend (`frontend/`)
- FastAPI backend (`backend/`)
- Audio content (`audio/`)
- Configuration & prompts (`config/`, `agent/`)

> A heartfelt blend of AI + human care for our elders.
